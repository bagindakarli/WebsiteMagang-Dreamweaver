minggu ke-2 (10 & 11 maret 2016)

convert scanned .pdf ke .doc dan di merge menggunakan able2extract

Naah, ini nih yang bikin gua bingung. Kerjaan gua di minggu kedua kali ini adalah pertama kalinya gua ngelakuin kerjaan ginian, yaitu Convert Scanned .pdf ke .doc dan dimerge menggunakan aplikasi Able2Extract (captionnya sengaja dibikin susah dimengerti, biar elite gitu). Jadi, awalnya ini gua disuruh bikin gimana caranya supaya file .pdf ini bisa di Search / Find pada saat file ini dibuka, sedangkan file .pdf ini adalah hasil Scaning buku-buku hardcopynya (foto-foto digabungin jadi file .pdf). Biasanya nih, kalo file .pdf itu hasil convert dari .doc nya langsung kan bisa di Search tuh pas filenya dibuka (misalnya kita bikin tulisan di Microsoft Word, terus langsung dijadiin .pdf). Nah, tapi kasus disini bener-bener mantep. Jadi, satu-satunya cara biar file .pdf hasil scaning-an ini bisa di Search yaaa diubah lah file ini menjadi sebuah teks, kemudian save lagi file tersebut menjadi file .pdf yang baru. Nah, di aplikasi ini (kalo beli/bayar/full/gak trial) bisa langsung satu file di convert jadi satu file .doc, tetapiiii.... sayangnya.... gua ga nemu crack (sebenernya ini dilarang si) atau serial key buat bikin aplikasi ini full. Jadi, sesuai aturan aplikasi bahwa "..Trial hanya bisa men-convert 3 halaman per file.." makanya proses merge gua pake dalam pemecahan masalah ini, hadeuhh capek mas nungguinnya. Tapi, setelah menunggu sekian lamanya, dan dengan tingkat kesabaran dan ketelitian yang tinggi, akhirnya selesai juga kerjaan di minggu kedua yang mantep ini... Alhamdulillah.
